Etiquetas THEAD, TBODY, TFOOT

cabecera de tabla
cuerpo de tabla
pie de tabla

Estas etiquetas no tienen ningún formato definido por defecto, por lo cual, debemos crearlo nosotros con hojas de estilo. Puede haber una o varias etiquetas tbody en una tabla, pero sólo una thead y una tfoot. Si una tabla tiene thead o tfoot, debe tener al menos un tbody.

Etiqueta UL

Esta etiqueta crea una lista no ordenada. el texto se publica con las viñetas de puntos grandes, pero no indica un orden.

  • Inicio
  • Noticias

Etiqueta TIME

Esta representa un periodo específico en el tiempo. Puede incluir el atributo datatime se utiliza para definir la fecha/hora en una forma legible por la máquina


año 2022 del mes de septiembre el dia 15 a las 10 40 minutos

Etiqueta SUP

La etiqueta define el texto de superíndice. El texto del superíndice aparece medio carácter por encima de la línea normal y, a veces, se representa en una fuente más pequeña. superíndice

Etiqueta TEMPLATE

La etiqueta se utiliza como contenedor para mantener contenido HTML oculto al usuario cuando se carga la página.template El contenido del interior se puede representar más tarde con un JavaScript

Etiqueta TEXTAREA

Esta etiquetaEl control asociado permite a los usuarios editar múltiples líneas de texto regular. Los controles textarea son útiles para recolectar o editar líneas largas de texto como mensajes, contenido de archivos, listas, reseñas, artículos,

Etiqueta SVG

Básicamente es un marcado, como HTML, excepto que tiene muchos elementos diferentes para definir las formas que desea que aparezcan en su imagen y los efectos que desea aplicar a esas formas. SVG es para marcar gráficos


Etiqueta TRACK

El elemento HTML track se utiliza como elemento hijo de los elementos multimedia, audio y video. Le permite especificar pistas de texto cronometradas (o datos basados en el tiempo), por ejemplo, para manejar subtítulos automáticamente.
